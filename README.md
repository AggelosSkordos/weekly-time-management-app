# Weekly Time Management App

A Python Object-Oriented Programming (OOP) application for weekly activity and time management.

## Features

- Create new users
- Save and load user data
- Add activities and tasks
- Edit existing activities
- Delete activities
- Track hours spent on activities
- Categorize activities
- Weekly time statistics
- Free time calculation
- Chore/productivity tracking
- Graphical statistics with matplotlib
- Data persistence using text files
- Input validation and error handling
- Menu-driven terminal interface

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- Matplotlib
- File Handling
- Git & GitHub

## Concepts Practiced

This project helped practice:

- Classes and objects
